# بوابة فواتير جرين توب

المشروع يحتوي على:

- صفحة عميل بسيطة تفتح الفاتورة من رابطها الخاص فقط.
- لوحة إدارة عربية لرفع الفواتير ونسخ الروابط وإيقافها وحذفها.
- تخزين خاص؛ لا توجد روابط عامة لملفات الفواتير.
- صلاحيات مقصورة على بريد الإدارة `info@greentaxikw.com`.
- إعدادات جاهزة للنشر على صفحات كلاودفلير.

## التجهيز لأول مرة

1. افتح محرر الأوامر في مشروع سوبابيس.
2. نفّذ الملف:
   `supabase/migrations/202607220001_secure_invoices.sql`
3. من قسم المستخدمين، أنشئ مستخدمًا واحدًا بالبريد:
   `info@greentaxikw.com`
4. انشر الوظيفة الموجودة في:
   `supabase/functions/get-invoice-url`
   مع إيقاف اشتراط رمز تسجيل الدخول؛ الملف `supabase/config.toml` يضبط ذلك تلقائيًا عند النشر بالأداة الرسمية.

الأوامر المختصرة عند استخدام أداة سوبابيس:

```bash
supabase link --project-ref mhyzbobdojfqyjbvgxow
supabase db push
supabase functions deploy get-invoice-url --no-verify-jwt
```

لا تضع مفتاح الخدمة السري داخل `config.js` أو داخل أي ملف في الموقع. المفتاح الموجود حاليًا مفتاح نشر عام مخصص للمتصفح، بينما مفتاح الخدمة يبقى داخل إعدادات سوبابيس فقط.

## النشر على كلاودفلير

اربط المستودع بصفحات كلاودفلير بهذه القيم:

- الفرع: `main`
- أمر البناء: `npm run build`
- مجلد الناتج: `dist`

بعد النشر:

- رابط العميل: `https://اسم-المشروع.pages.dev/?token=...`
- لوحة الإدارة: `https://اسم-المشروع.pages.dev/admin`

## اختبار المشروع

```bash
npm test
```
